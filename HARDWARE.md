# Hardware Specification

## Server
- **Hostname:** beast-node2
- **OS:** Ubuntu, Linux 6.8.0-60-generic (x86_64)

## CPU
- **Model:** Intel Xeon Gold 6230 @ 2.10 GHz (Turbo up to 3.90 GHz)
- **Sockets:** 2
- **Cores per socket:** 20
- **Threads per core:** 2 (Hyperthreading)
- **Total logical CPUs:** 80
- **Architecture:** Cascade Lake (Stepping 7)

## Memory
- **Total RAM:** 188 GB DDR4
- **Swap:** 8 GB

## Storage
- **Boot/OS disk:** Intel SSD SC2KB96 (894 GB, SATA SSD)
- **Data disk (`/data`):** Intel SSD PE2KE032T8 (2.9 TB, NVMe)
  - PCIe NVMe SSD
  - Used for all database data directories
